Manual Testing Checklist

Desktop Navigation (Screen width > 768px):

  • ✓ Logo is clickable and navigates to home
  • ✓ All navigation links are clickable
  • ✓ Hover effects work properly
  • ✓ Focus states are visible for keyboard navigation
  • ✓ No overlapping elements blocking clicks

Mobile Navigation (Screen width ≤ 768px):

  • ✓ Hamburger menu button is visible and clickable
  • ✓ Mobile menu opens when hamburger is clicked
  • ✓ Mobile menu closes when overlay is clicked
  • ✓ Mobile menu closes when a link is clicked
  • ✓ Mobile menu closes when Escape key is pressed
  • ✓ Navigation links in mobile menu are clickable
